# VUE框架

## 介绍
[VUE](https://vuejs.org/) 是一个用于构建用户界面的渐进式 JavaScript 框架。与其他大型框架不同，Vue 被设计为可以自底向上逐层应用。
Vue 的核心库只关注视图层，易于上手，并与其他库或现有项目进行整合。此外，Vue 也完全有能力驱动复杂的单页应用程序。

## vue的一些方法
ref 和 reactive 的区别：
- ref 用于创建一个响应式的基本数据类型（如字符串、数字、布尔值等）或对象引用，而 reactive 用于创建一个响应式的对象。
- ref 返回一个包含 .value 属性的对象，而 reactive 返回一个代理对象，直接操作其属性即可。

v-on指令：
- v-on 指令用于监听 DOM 事件并在触发时执行一些 JavaScript代码。它的简写形式是 @。

v-show 和 v-if 的区别：
- v-if 是条件渲染，只有在条件为真时才会渲染元素，条件为假时元素会被完全移除。
- v-show 是通过 CSS 的 display 属性来控制元素的显示和隐藏，元素始终会被渲染到 DOM 中。
可以通过一些属性的设定来控制元素的显示与隐藏。

v-bind 指令：
- v-bind 指令用于动态绑定元素的属性或组件的 props。它的简写形式是 :。
例如v-bind:href 可以简写为 :href。

v-for 指令：
- v-for 指令用于基于一个数组或对象来渲染一个列表。可以用来遍历数组或对象
